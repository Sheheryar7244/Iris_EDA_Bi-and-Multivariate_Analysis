🌸 Iris Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project explores the Iris dataset using EDA techniques to understand relationships between different flower features and species classification.

---
📊 Key Objectives
- Analyze feature distributions
- Study relationships between variables
- Visualize class separability
- Identify patterns in species classification

---
🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

---
📈 Analysis Performed

🔹 Univariate Analysis
- Histograms
- Count plots

🔹 Bivariate Analysis
- Scatter plots (feature comparisons)
- Box plots

🔹 Multivariate Analysis
- Pair plots
- Correlation heatmap
- Pivot tables

---
📊 Visualizations Included
- 📊 Count Plot
- 📉 Histogram
- 📦 Box Plot
- 🔵 Scatter Plot
- 🔗 Pair Plot
- 🔥 Heatmap
